# Phonics Learning — Ultralearning Project

A focused 4-week project to build a reliable, automatic system for sounding out unfamiliar English words and foreign names when reading aloud — especially in professional and work contexts.

## Goal

By **July 20, 2026**, read aloud unfamiliar words and names with confidence, using syllable rules and prefix/suffix patterns instead of guessing or skipping.

## Timeline

| Item | Detail |
|------|--------|
| **Start** | June 23, 2026 |
| **End** | July 20, 2026 |
| **Duration** | 4 weeks |
| **Daily time** | 15–20 minutes |

## Daily Routine (15–20 min)

1. **Drill** (5–7 min) — targeted phonics/syllable practice from `drills/`
2. **Reading aloud** (7–10 min) — apply skills to real text (articles, names, work docs)
3. **Log** (2–3 min) — record session in `sessions/` using the template
