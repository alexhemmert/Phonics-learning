## Day 1 Detail — VCCV Rule

**Rule:** When you see **Vowel–Consonant–Consonant–Vowel**, split between the two consonants.

| Word | Split | Pronunciation hint |
|------|-------|--------------------|
| hap-py | hap \| py | Short vowel in first syllable |
| let-ter | let \| ter | |
| win-ter | win \| ter | |
| pic-nic | pic \| nic | |
| con-tract | con \| tract | |

**Practice flow:** Say rule → split 5 words aloud → read paragraph applying rule.

---

## Day 2 Detail — Flexible VCV Rule

**Rule:** When you see **Vowel–Consonant–Vowel**, the split depends on whether the first syllable is "closed" (ends in consonant → short vowel) or "open" (ends in vowel → long vowel).

| Pattern | Split | Example |
|---------|-------|---------|
| Closed first syllable | VC \| V | ro-bot, cab-in, lev-el |
| Open first syllable | V \| CV | ba-by, ti-ger, pi-lot |

**When unsure:** Try splitting after the consonant first (closed). If it sounds wrong, try before (open).

## Prefix Quick Reference (Days 3–6)

| Prefix | Sound | Example |
|--------|-------|---------|
| un- | /ʌn/ | unhappy → un \| hap \| py |
| re- | /ri/ or /riː/ | rewrite → re \| write |
| pre- | /pri/ | preview → pre \| view |
| dis- | /dɪs/ | disagree → dis \| a \| gree |
| in- / im- | /ɪn/ or /ɪm/ | impossible → im \| pos \| si \| ble |

*Prefixes are their own syllable — split them off first, then apply syllable rules to the root.*
