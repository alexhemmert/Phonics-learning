# Ultralearning Map — Phonics Learning

> **Living document.** Update weekly focus, benchmarks, and experiment log as you learn.

## Project Goal

Build a **reliable, automatic** habit for sounding out unfamiliar English words and foreign names when reading aloud — without pausing, guessing, or avoiding difficult words in professional settings.

**Success looks like:** You encounter an unfamiliar word in a meeting prep doc or article, split it into syllables using known rules, pronounce it confidently, and move on — all in under 5 seconds.

## Skill Hierarchy

Prioritized skills for this project. Higher priority = build first.

| Priority | Skill | Description | Week |
|----------|-------|-------------|------|
| P0 | Syllable splitting (VCCV) | Split between two consonants: *hap-py*, *let-ter* | 1 |
| P0 | Syllable splitting (VCV) | Split before or after single consonant: *ro-bot* vs. *ba-by* | 1 |
| P1 | Common prefixes | Decode meaning + pronunciation: un-, re-, pre-, dis-, in- | 1–2 |
| P1 | Common suffixes | -tion, -sion, -ment, -able, -ous | 2 |
| P2 | Vowel teams | ai, ea, oo, ough patterns | 2–3 |
| P2 | Silent letters & irregulars | kn-, -mb, -ght, common exceptions | 3 |
| P3 | Foreign name patterns | Slavic, Germanic, Romance name stress | 3–4 |
| P3 | Speed & automaticity | < 5 sec per unfamiliar word under pressure | 4 |
